
# 📘 Tarefa: Jogo da Forca

## 🎯 Objective

Aprenda a construir um jogo interativo em Python usando strings, loops, condicionais e entrada de dados do usuário. O objetivo é praticar lógica de programação e fluxo de controle enquanto cria um jogo simples e divertido.

## 📝 Tasks

### 🛠️ Implementação do Jogo da Forca

#### Descrição
Crie um jogo da Forca em que o programa seleciona uma palavra aleatória e o jogador tenta adivinhar letra por letra até completar a palavra ou esgotar as chances.

#### Requisitos
O programa completo deve:

- Selecionar palavras aleatoriamente de uma lista predefinida.
- Aceitar palpites de letras do usuário e mostrar o progresso atual da palavra (por exemplo, `_ _ _ _`).
- Rastrear tentativas incorretas restantes e limitar o número de erros permitidos.
- Encerrar o jogo quando a palavra for adivinhada ou as tentativas se esgotarem.
- Exibir mensagens de vitória ou derrota de forma clara.

### 🛠️ Validação de Entrada e Interação

#### Descrição
Garanta que o jogo lide corretamente com a entrada do usuário e ofereça feedback amigável durante a partida.

#### Requisitos
O programa completo deve:

- Ignorar espaços em branco extras e diferença entre letras maiúsculas e minúsculas.
- Avisar o jogador se a letra já foi digitada.
- Mostrar o número de tentativas restantes após cada palpite.
- Permitir que o jogador continue jogando até o final do jogo com as condições corretas de término.

### 🛠️ Estilo e Organização do Código

#### Descrição
Organize o código em funções claras e fáceis de entender, mantendo o programa modular.

#### Requisitos
O programa completo deve:

- Incluir pelo menos uma função para selecionar a palavra e outra para processar os palpites.
- Manter o fluxo principal do jogo em um bloco `if __name__ == "__main__":`.
- Comentar trechos importantes, explicando a lógica quando necessário.
- Usar nomes de variáveis descritivos para facilitar a leitura e manutenção.
